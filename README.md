# Progress Logs — How To Use

Each person logs daily progress in their own file:
- docs/person1-backend/LOG.md
- docs/person2-ai/LOG.md
- docs/person3-frontend/LOG.md
- docs/person4-devops/LOG.md

Only add entries to YOUR OWN file. Never edit or delete someone else's entries.

## Format (copy this for every new entry):

## [Month].[Week].[Day] — [Date]
**Did:** one line, what you actually did today
**Result:** one line, what you ended up with (working / broken / partial)
**Notes:** (optional) anything weird you hit, skip if nothing
